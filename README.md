# CODSOFT Java Development Internship

This repository contains the Java Development Internship tasks completed as part of the CodSoft Internship Program.

## Tasks

### Task 1 - Number Game
A Java console application where the user guesses a randomly generated number with limited attempts.

### Task 2 - Student Grade Calculator
A Java program that calculates total marks, average percentage, and grade based on user input.

### Task 3 - ATM Interface
A simple ATM system using Object-Oriented Programming (OOP) that allows users to check balance, deposit, and withdraw money.

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- VS Code
- Git & GitHub
